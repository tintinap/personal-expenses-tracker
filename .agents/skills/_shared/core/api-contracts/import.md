## Endpoints

### POST /import/transactions
- **Description**: Import multiple transactions parsed from Excel (Web import flow)
- **Auth**: Required
- **Request Body**:
  ```json
  {
    "transactions": [
      {
        "date": "string (YYYY-MM-DD, required)",
        "transactionType": "string (expense | currency_income | currency_exchange_out | currency_exchange_in, required)",
        "note": "string (optional, max 200 chars)",
        "categoryName": "string (optional, required for expense)",
        "originalAmount": "number (required, > 0)",
        "originalCurrency": "string (required, 3-char ISO code)",
        "amountBase": "number (optional)",
        "exchangeRate": "number (optional)",
        "rateSource": "string (optional)",
        "id": "string (UUID, optional)",
        "exchangeEventId": "string (UUID, optional)",
        "sourceLabel": "string (optional)",
        "isAggregate": "boolean (optional, default: false)"
      }
    ]
  }
  ```
- **Response 201**:
  ```json
  {
    "success": true,
    "importedCount": 115
  }
  ```
- **Errors**:
  - `400 Bad Request`: Validation failure (e.g. invalid date format, missing category for expense, negative amount)
  - `401 Unauthorized`: Missing or invalid JWT token
  - `500 Internal Server Error`: Server failure

## Data Models

### ImportTransaction
| Field | Type | Required | Description |
|---|---|---|---|
| id | UUID | no | If matching existing transaction in DB, it is updated. If missing, auto-generated |
| date | string | yes | YYYY-MM-DD format |
| transactionType | string | yes | `expense`, `currency_income`, `currency_exchange_out`, `currency_exchange_in` |
| note | string | no | Max 200 chars |
| categoryName | string | no | Category name; required for expense; must match existing category |
| originalAmount | number | yes | Decimal amount, positive |
| originalCurrency | string | yes | 3-char ISO currency code (e.g. AUD, USD) |
| amountBase | number | no | Equivalent in base currency; calculated if omitted |
| exchangeRate | number | no | Exchange rate; default 1.0 (same currency) or fetched if omitted |
| rateSource | string | no | Default: `import` |
| exchangeEventId | UUID | no | Grouping id for currency exchanges |
| sourceLabel | string | no | Source label for income |
| isAggregate | boolean | no | Default: false; when true, represents a period-level aggregate |
